# Restaurant-Website-UI
A simple and clean "multi-page restaurant website" created using "HTML and CSS".  
This project focuses on layout design, page structure, and basic styling.

---

# Pages Included
- Home
- About Us
- Our Menu
- Gallery
- Contact

---

# Technologies Used
- HTML5
- CSS3
- Font Awesome (icons)

---

# How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Use the navigation menu to explore pages
